Step 1: Create GA4 Property

  1. Go to Google Analytics

  3. Click "Create Account" or "Admin" (gear icon)
  4. Fill in:
    • Account name: "Oscar Espinoza Photography"

    • Business size: "Small"
  5. For Data Stream:
    • Platform: Web
    • Website URL: https://food.oscarespinoza.co.uk (your domain)
    • Stream name: "Food Photography Site"

Step 2: Get Your Measurement ID

After creating the property, you'll see a Measurement ID like:

G-XXXXXXXXXX

Copy this ID - you'll need it for the next step!

Step 3: Add Your Measurement ID

  1. Open file: includes/analytics.php
  2. Find line 4: $GA4_MEASUREMENT_ID = 'G-XXXXXXXXXX';
  3. Replace G-XXXXXXXXXX with your actual Measurement ID
  4. Save the file
