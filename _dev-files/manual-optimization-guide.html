Step 2: Create Optimized Versions

For each image in assets/hero-images/:

  1. Upload to Squoosh.app
  2. Create these versions:
    • WebP format: Quality 80, resize to max 1200px wide
    • JPEG format: Quality 80, resize to max 1200px wide
    • Optional: AVIF format for even smaller files
  3. Save to: assets/optimized/ folder
  4. Naming: Keep same filename but change extension
    • photo.jpg ‚Üí photo.webp
    • photo.jpg ‚Üí photo.jpg (optimized)

Step 3: Quick Batch Processing with Squoosh

  1. Go to: squoosh.app
  2. Drag your first image from assets/hero-images/
  3. Set right panel to:
    • Format: WebP
    • Quality: 80
    • Resize: Max width 1200px
  4. Download and save to assets/optimized/
  5. Repeat for JPEG format
  6. Continue with next image

Step 4: Expected Results

Before: ~2MB JPEG files

After:

  • WebP: ~400-600KB (70% smaller)
  • Optimized JPEG: ~600-800KB (60% smaller)
  • AVIF: ~300-400KB (80% smaller)

Performance improvement: Pages will load 3-5x faster!

Step 5: Test Your Optimized Site

Once you've optimized a few images:

  1. Refresh your website at localhost:8000
  2. Open browser DevTools (F12) ‚Üí Network tab
  3. Reload page and check image sizes
  4. You should see much smaller file sizes!
