📘 Meta Pixel Setup - Facebook/Instagram Retargeting

Turn website visitors into photography clients! Meta Pixel lets you show targeted ads to people who visited your site but didn't book yet.

Step 1: Create Meta Pixel

  1. Go to Facebook Events Manager
  2. Click "Create Pixel"
  3. Name it: "Oscar Espinoza Food Photography"
  4. Enter your website: https://food.oscarespinoza.co.uk
  5. Copy your Pixel ID (looks like: 123456789012345)

Step 2: Add Your Pixel ID

  1. Open file: includes/meta-pixel.php
  2. Find line: $META_PIXEL_ID = 'YOUR_PIXEL_ID';
  3. Replace YOUR_PIXEL_ID with your actual Pixel ID
  4. Save the file
💡 Example:
$META_PIXEL_ID = '123456789012345';

Step 3: Create Custom Audiences

In Facebook Ads Manager, create these audiences for retargeting:

🎯 High-Intent Visitors

  • Event: InitiateCheckout (clicked Stripe buttons)
  • Time window: 30 days
  • Use for: "Book a shoot" ads

📧 Lead Warm-Up

  • Event: Lead (submitted contact form)
  • Time window: 180 days
  • Use for: Portfolio showcase ads

🖼️ Portfolio Viewers

  • Event: ViewContent on portfolio pages
  • Time window: 30 days
  • Use for: Service promotion ads

⏰ Service Page Visitors

  • Event: Search (spent 30+ seconds on services)
  • Time window: 7 days
  • Use for: Urgency/discount ads

Step 4: Create Lookalike Audiences

Once you have conversions, create lookalike audiences:

  • Source: People who paid deposits (InitiateCheckout)
  • Location: United Kingdom
  • Size: 1% (most similar)
  • Use for: Finding new photography clients

Step 5: Track Your Results

Monitor these metrics in Facebook Ads Manager:

  • 📧 Lead Events: Contact form submissions
  • 💳 InitiateCheckout: Stripe deposit clicks
  • 📞 Schedule Events: Consultation bookings
  • 🖼️ ViewContent: Portfolio engagement
  • 🔍 Search Events: High-intent visitors

🔍 Testing Mode

On localhost:8000, Meta Pixel events are logged to browser console. Open DevTools → Console to see:
